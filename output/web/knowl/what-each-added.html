What Each Extension Added.

  • \(\N\text{:}\) counting numbers

  • \(\Z\text{:}\) added zero and negatives (subtraction always possible)

  • \(\Q\text{:}\) added fractions (division always possible, except by zero)

  • \(\R\text{:}\) added irrationals like \(\sqrt{2}, \pi, e\) (filled gaps)

  • \(\C\text{:}\) added imaginary numbers (every polynomial solvable)
