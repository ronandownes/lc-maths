Preface How to Use This Book

This book covers the Ordinary Level learning outcomes for the Leaving Certificate Mathematics course. The curriculum is organized into five strands:

  1. Statistics and Probability

  2. Geometry and Trigonometry

  3. Number

  4. Algebra

  5. Functions

Each chapter corresponds to one strand and contains all the learning outcomes specified in the Leaving Certificate curriculum.
